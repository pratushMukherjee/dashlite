# DashLite Demo Queries

## Search Queries (try on /search)
- `conflict resolution` — finds architecture notes + API spec + security review
- `embedding model evaluation` — finds experiment results + ML pipeline design
- `security vulnerabilities` — finds security review document
- `quarterly revenue` — finds quarterly report
- `Kubernetes deployment` — finds architecture notes + meeting notes

- `What are the main security concerns found in the review?`
- `Which embedding model was recommended and why?`
- `What are the top user complaints from the quarterly report?`
- `How does the hybrid search system work?`
